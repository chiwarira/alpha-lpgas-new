  • Column 2: Description/Reference (client name, address, or invoice number)
  • Column 3: Amount
  • Column 4: Running Balance

How It Works
  1. Upload your bank statement CSV file
  2. The system analyzes each transaction and finds matching unpaid invoices based on:
    • Invoice number in description
    • Client name similarity
    • Address match
    • Amount match
  3. Review suggested matches and accept or reject them
  4. Accepted matches automatically create EFT payment records and mark invoices as paid
Note: Only transactions dated on or after the invoice issue date will be matched.
